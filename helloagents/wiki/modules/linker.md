# 链接器

## 目的
将目标文件链接为可执行或 COE 文件。

## 模块概述
- **职责:** 符号解析、段布局、COE 生成
- **状态:** ✅稳定
- **最后更新:** 2026-01-16

## 规范

### 需求: 目标文件链接
**模块:** 链接器
支持将多个 `.o` 文件链接为 `.elf` 或 `.coe`。

#### 场景: 输出 COE
生成 Vivado 可用的 COE 初始化文件。
- 预期结果: 生成 COE 文件并通过校验

## API接口
- CLI 入口由外层工具提供，模块内无对外 API

## 数据模型
- COE 文件

## 依赖
- assembler
