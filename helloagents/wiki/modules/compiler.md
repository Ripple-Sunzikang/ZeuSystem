# 编译器

## 目的
将 C 子集代码编译为 RISC-V 汇编与中间表示。

## 模块概述
- **职责:** 词法/语法/语义分析，IR 生成与代码生成
- **状态:** ✅稳定
- **最后更新:** 2026-01-16

## 规范

### 需求: 编译 C 到汇编
**模块:** 编译器
支持 C 子集语法并输出 RISC-V 汇编。

#### 场景: 基础程序编译
提供单文件与多文件编译，输出汇编或 COE。
- 预期结果: 生成汇编/COE 文件

## API接口
- CLI 入口由外层工具提供，模块内无对外 API

## 数据模型
- 编译中间产物: IR 文本与汇编文本

## 依赖
- assembler
- linker
