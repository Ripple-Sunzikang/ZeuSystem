# 汇编器

## 目的
将 RISC-V 汇编转换为 ELF/机器码。

## 模块概述
- **职责:** 汇编解析、编码与目标文件生成
- **状态:** ✅稳定
- **最后更新:** 2026-01-16

## 规范

### 需求: 汇编到目标文件
**模块:** 汇编器
支持汇编输入并输出 ELF 目标文件。

#### 场景: 基础汇编
输入 `.s` 文件输出 `.o` 文件。
- 预期结果: 生成可链接的 ELF 目标文件

## API接口
- CLI 入口由外层工具提供，模块内无对外 API

## 数据模型
- ELF 目标文件
