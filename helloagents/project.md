# 项目技术约定

---

## 技术栈
- **核心:** Rust (CLI/编译器) / Verilog (硬件) / Vivado 2017.4
- **辅助:** Bash 脚本 / Electron (IDE)

---

## 开发约定
- **代码规范:** Rust 使用 rustfmt 默认风格；脚本保持 POSIX 兼容优先
- **命名约定:** 文件名与目录名使用小写加下划线或短横线

---

## 错误与日志
- **策略:** CLI 统一返回非零退出码并输出清晰错误信息
- **日志:** 关键步骤输出简要状态与路径信息

---

## 测试与流程
- **测试:** 以功能验证与手工流程为主，必要时补充脚本测试
- **提交:** Commit message 使用简明动词开头的英文短语
