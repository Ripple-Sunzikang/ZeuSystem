# Changelog

本文件记录项目所有重要变更。
格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/),
版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [Unreleased]
### 修复
- IDE 取消任务会终止 CLI 进程及其子进程（包含 Vivado 综合/实现/烧录）。
- IDE 取消任务改为递归终止子进程树，避免 Vivado 残留。
- IDE 任务日志增加 PID 与取消信号提示，便于排查中断来源。
### 变更
- Vivado 流程拆分为综合与实现，新增 `seu synth`/`seu impl`（`bitstream` 为别名）。
