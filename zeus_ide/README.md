# ZEUS IDE 文档入口

ZEUS IDE 是基于 Electron 的配套开发环境，提供 C 代码编辑、编译、输出查看等功能。

## 快速入口

- `ZEUS_IDE_Complete_Manual.md` - 完整用户手册
- `ZEUS_IDE_Deployment_Guide.md` - 部署与构建
- `ZEUS_IDE_Feature_Documentation.md` - 功能与实现细节

## 基本运行

```bash
npm install
npm start
```

> 注意：IDE 内部包含 Windows 路径示例与默认查找逻辑，若你的仓库路径不同，请在 IDE 中手动指定编译器路径。
