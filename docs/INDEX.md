## ✨ 文档约定

所有文档遵循以下约定：

1. **Markdown格式** - 所有文档均采用Markdown格式
2. **UTF-8编码** - 支持中英文混合
3. **清晰结构** - 使用标题、列表、表格组织内容
4. **代码示例** - 包含可执行的代码示例
5. **目录导航** - 长文档包含目录导航
